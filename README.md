# Penguin Adventure Demo

A small 2D platformer game built with Unity 6.

Play as a penguin character, jump across platforms, avoid obstacles, and explore a pixel-art world.

## Features

* 2D platformer gameplay
* Pixel art style
* WebGL build support
* Basic movement & jump mechanics
* AI-generated sprites and assets created/customized by myself

## Controls

| Key        | Action |
| ---------- | ------ |
| Arrow Keys | Move   |
| Space      | Jump   |

## Tech Stack

* Unity 6
* C#
* WebGL
